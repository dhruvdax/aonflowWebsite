# **App Name**: FlowSync

## Core Features:

- Navigation Bar: Interactive navigation bar with smooth transitions between pages.
- Contact Form: Submission form for user inquiries, forwarding details to a predefined email address.
- Responsive Design: Fully responsive layout adapting seamlessly across different devices.
- Theme Toggle: Light/dark mode toggle for user interface customization.
- Content Generation Tool: AI powered tool to automatically generate page descriptions based on page titles using web retrieval, adapting writing style and length depending on the website.

## Style Guidelines:

- Primary color: Deep blue (#264FDA), mirroring Aonflow's palette, evoking trust and professionalism.
- Background color: Very light gray (#F5F5F5) to provide a clean, modern backdrop in light mode.
- Accent color: Soft purple (#5172e1) for highlights and interactive elements, ensuring a modern look.
- Body and headline font: 'Poppins', a grotesque-style sans-serif with a neutral, objective, modern look; suitable for both headlines and body text.
- Crisp, minimalist icons to complement the modern aesthetic.
- Clean, grid-based layout inspired by Aonflow.com, ensuring a professional and easy-to-navigate user experience.
- Subtle transitions and animations to enhance user interaction without being distracting.
